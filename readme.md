### Запуск

1. python3 -m venv menu
2. cd menu
3. source bin/activate
4. git clone https://github.com/dociw/menu.git
5. pip install -r requirements.txt
6. cd menu
7. ./manage.py runserver
8. перейти в браузере по адресу - http://localhost:8000
